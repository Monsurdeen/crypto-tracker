# 📈 CryptoTrack Pro

A real-time cryptocurrency tracking dashboard built with React and Vite.

## Live Demo
[crypto-tracker.vercel.app](https://crypto-tracker.vercel.app)

## Features
- Live prices for top 50 coins via CoinGecko API
- 7-day sparkline chart per coin
- 30-day price history chart on coin detail
- Real-time search and filter
- Global market stats (market cap, volume, BTC dominance)
- Auto-refresh every 60 seconds
- Dark / light mode toggle
- Fully responsive

## Tech Stack
- React 18 + Vite
- Recharts
- Axios
- CoinGecko Public API

## Run Locally
git clone https://github.com/Monsurdeen/crypto-tracker.git
cd crypto-tracker
npm install
npm run dev
